# 🚀 Space Math 


## 🌌 Overview

**Space Math** is an interactive educational game built with Python’s Turtle graphics module. Players solve math problems in a space-themed environment and earn celestial rewards based on their performance:

* ☀️ **Sun** → Correct answer
* 🌙 **Moon** → Incorrect answer

The goal is to make learning math fun, visual, and engaging - especially for beginners and young learners.


## 🎮 Features

* 🧮 Randomly generated math questions
* 🌠 Space-themed visual interface using Turtle graphics
* 🏆 Reward system (Sun & Moon tracking)
* 🎯 Immediate feedback on answers
* 🕹️ Simple controls and user-friendly experience
* 🤔 Difficulty levels (Easy, Medium, Hard)


## 🧠 Learning Objectives

This game helps users:

* Practice basic arithmetic (addition, subtraction, multiplication, division)
* Improve mental math speed and accuracy
* Build confidence in problem-solving


## 🛠️ Technologies Used

* **Python 3**
* **Turtle Graphics Module**


## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/math-in-space.git
cd math-in-space
```

### 2. Ensure Python is Installed

Check your Python version:

```bash
python --version
```

Python 3.x is required.

### 3. Run the Game

```bash
python space_math.py
```


## ▶️ How to Play

1. Launch the game.
2. A math question will appear on the screen.
3. Enter your answer using the keyboard.
4. Press **Enter** to submit.
5. Receive feedback:

   * Correct → ☀️ Sun added
   * Incorrect → 🌙 Moon added
6. Continue solving questions and track your score.


## 🧩 Game Mechanics

* Questions are randomly generated each round.
* Score is tracked visually using icons (Sun/Moon).
* Game continues until a user exits.
